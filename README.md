# 🐴 海马王国 (Seahorse Kingdom)

一个可爱的海马养成模拟游戏网页应用。

## 🌟 特性

- **养成系统** - 精心照料你的海马，喂食、换水、逗玩
- **繁殖系统** - 配对繁殖，培育优质海马后代
- **任务系统** - 每日任务获取奖励
- **商城系统** - 兑换海马相关商品
- **个人中心** - 查看资产、订单、邀请好友

## 📱 预览

打开 `seahorse-kingdom.html` 即可在浏览器中运行游戏。

功能页面：
- 🏠 首页 - 总览资产和功能入口
- 🐴 养成 - 管理和升级你的海马
- ❤️ 繁殖 - 配对繁殖获得新海马
- 🛒 商城 - 兑换商品
- 👤 我的 - 个人中心

## 🎮 如何游玩

1. 打开游戏首页
2. 点击「养成」照顾你的海马
3. 完成任务获取海马券
4. 在「繁殖」页面配对海马
5. 去「商城」兑换心仪商品
